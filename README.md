# Ben's Dotfiles

Ben's dotfiles (all in one place!)

## Setup

To setup all dotfiles on the system

```sh
git clone https://github.com/bitHero/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./bootstrap
```
